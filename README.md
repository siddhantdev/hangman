# Hangman

A simple hangman game written in Java using the Swing Library.

## Running the game

Clone this repository

~~~bash
git clone https://github.com/siddhantdev/hangman.git
~~~

cd into src

~~~bash
cd hangman/src
~~~

Compile the game with:

~~~bash
javac Main.java
~~~

Run the game using the command:

~~~bash
java Main
~~~
